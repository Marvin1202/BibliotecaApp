# BibliotecaApp

## Descripción
Esta es una aplicación de escritorio hecha en C# para manejar libros, usuarios y préstamos de una biblioteca.  
Permite agregar, actualizar y eliminar registros, además de registrar préstamos y devoluciones.

## Instalación
1. Clonar el repositorio a tu computadora.
2. Abrir el proyecto en Visual Studio.
3. Compilarlo y ejecutar la aplicación.

## Uso
- **Libros**: puedes agregar libros nuevos, modificarlos o eliminarlos.  
- **Usuarios**: puedes registrar nuevos usuarios, actualizar sus datos o eliminarlos.  
- **Préstamos**: registrar préstamos, devolver libros y ver si están disponibles.  
- **Gráficos**: ver un gráfico que muestra los libros que se prestan más seguido.

## Guía de despliegue
1. Asegúrate de tener Visual Studio instalado.  
2. Abrir el archivo `BibliotecaApp.sln`.  
3. Ejecutar con F5 o Ctrl+F5.  
4. La aplicación abrirá la interfaz con pestañas para libros, usuarios, préstamos y un gráfico de los libros más prestados.

## Flujo de Git
- `main`: contiene la versión final del proyecto lista para entregar.  
- `develop`: aquí se integran todas las funcionalidades antes de pasarlas a main.  
- `feature/libros`, `feature/usuarios`, `feature/prestamos`: aquí se trabajó cada parte del proyecto por separado.

> Nota: Todos los primeros cambios los hice en main y después creé las ramas para practicar cómo se trabaja con Git y organizar mejor el proyecto.
